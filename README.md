#### Функциональные требования
- Поддержка базовых конструкций:
  - Арифметика: `+`, `-`, `*`, `/`, `%`
  - Условия: `if`, `else`
  - Циклы: `for`, `while`
  - Функции и рекурсия
- Блоки кода обозначаются `{}`  
- Комментарии: `// комментарий`
- Идентификаторы состоят из латинских букв, цифр и `_`

#### Пример кода
```text
func fact(n) {
    if n <= 1 {
        return 1
    }
    return n * fact(n - 1)
}
